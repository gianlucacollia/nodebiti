════════════════════════════════════════════════════════════
🚀 METODO 1: GITHUB WEB (PIÙ SEMPLICE) ⭐ CONSIGLIATO
════════════════════════════════════════════════════════════

Carica direttamente dal browser, NESSUN software da installare!

──────────────────────────────────────────────────────────
PASSO 1: Crea Account GitHub (se non ce l'hai)
──────────────────────────────────────────────────────────

1. Vai su: https://github.com
2. Clicca "Sign up"
3. Registrati con email e password
4. Conferma email

──────────────────────────────────────────────────────────
PASSO 2: Crea Nuovo Repository
──────────────────────────────────────────────────────────

1. Fai login su GitHub
2. Clicca il "+" in alto a destra
3. Scegli "New repository"

4. Compila:
   - Repository name: nodebiti-it
   - Description: Sito web per assistenza gestione debiti
   - ✅ Public (visibile a tutti)
   - ✅ Add a README file
   - Add .gitignore: None
   - License: MIT License (opzionale)

5. Clicca "Create repository"

──────────────────────────────────────────────────────────
PASSO 3: Carica i File
──────────────────────────────────────────────────────────

1. Nella pagina del repository, clicca "Add file"
2. Scegli "Upload files"

3. Trascina questi file dalla cartella nodebiti-ottimizzato:
   
   FILE ESSENZIALI (da caricare sempre):
   ☑ index.html
   ☑ styles.css
   ☑ app.js
   
   FILE OPZIONALI (utili ma non necessari):
   ☑ README.md (se vuoi)
   ☑ CONFIG-EMAIL.txt
   ☑ GUIDA-PUBBLICAZIONE.txt
   
   FILE DA NON CARICARE:
   ✗ index-standalone.html (è una copia)
   ✗ LEZIONE-BASI-SITI-WEB.txt (solo per te)
   ✗ GUIDA-COMPLETA-ONLINE.txt (solo per te)

4. Scrivi commit message: "Initial commit - NoDebiti.it v1.0"
5. Clicca "Commit changes"

FATTO! Il tuo codice è su GitHub! 🎉

──────────────────────────────────────────────────────────
PASSO 4: Attiva GitHub Pages (BONUS - Sito Online!)
──────────────────────────────────────────────────────────

1. Nel tuo repository, vai su "Settings"
2. Nel menu laterale, clicca "Pages"
3. Sotto "Source", seleziona "main" branch
4. Clicca "Save"

Aspetta 1-2 minuti e il tuo sito sarà online su:
https://tuo-username.github.io/nodebiti-it

COMPLETAMENTE GRATIS! 🎉

──────────────────────────────────────────────────────────

════════════════════════════════════════════════════════════
⚡ METODO 3: GIT COMMAND LINE (PIÙ TECNICO)
════════════════════════════════════════════════════════════

Per chi vuole usare git da terminale.

──────────────────────────────────────────────────────────
PASSO 1: Installa Git
──────────────────────────────────────────────────────────

1. Scarica: https://git-scm.com/download/win
2. Installa con impostazioni default
3. Riavvia PowerShell/CMD

──────────────────────────────────────────────────────────
PASSO 2: Configura Git (prima volta)
──────────────────────────────────────────────────────────

git config --global user.name "Tuo Nome"
git config --global user.email "tuaemail@example.com"

──────────────────────────────────────────────────────────
PASSO 3: Inizializza Repository
──────────────────────────────────────────────────────────

cd C:\Users\gianl\nodebiti-ottimizzato
git init
git add .
git commit -m "Initial commit - NoDebiti.it v1.0"

──────────────────────────────────────────────────────────
PASSO 4: Crea Repository su GitHub
──────────────────────────────────────────────────────────

1. Vai su github.com
2. Crea nuovo repository "nodebiti-it"
3. NON aggiungere README/gitignore/license

──────────────────────────────────────────────────────────
PASSO 5: Collega e Carica
──────────────────────────────────────────────────────────

git remote add origin https://github.com/TUO-USERNAME/nodebiti-it.git
git branch -M main
git push -u origin main

Ti chiederà username e password GitHub.
FATTO! 🎉

📝 CREA UN README.md
════════════════════════════════════════════════════════════

Crea un file README.md con questo contenuto:

─────────────────────────────────────────────────────────
# 🐇 NoDebiti.it

Piattaforma web per assistenza nella gestione dei debiti.

## 🌟 Caratteristiche

- Wizard interattivo a 3 step
- Calcolo automatico totale debiti
- Database di 50+ professionisti
- Ricerca e filtri avanzati
- Invio email automatico con EmailJS
- Design responsive e accessibile

## 🚀 Demo Live

[Vedi il sito live](https://tuo-username.github.io/nodebiti-it)

## 💻 Tecnologie

- HTML5
- CSS3 (variabili CSS, Flexbox, Grid)
- JavaScript (ES6+)
- EmailJS per invio email

## 📦 Installazione

1. Clona il repository
2. Apri `index.html` nel browser
3. (Opzionale) Configura EmailJS seguendo `CONFIG-EMAIL.txt`

## 🎨 Personalizzazione

### Colori

Modifica le variabili CSS in `styles.css`:

```css
:root {
  --bg-primary: #f0f4f8;
  --accent: #5dade2;
  /* ... */
}
```

### Professionisti

Modifica l'array `PROFESSIONALS_DATA` in `app.js`

## 📧 Configurazione Email

Vedi `CONFIG-EMAIL.txt` per istruzioni complete su EmailJS.

## 📄 Licenza

MIT License - Usa liberamente per i tuoi progetti!

## 👤 Autore

Creato con ❤️ per aiutare le persone a gestire i debiti
