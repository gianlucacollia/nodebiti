════════════════════════════════════════════════════════════
1️⃣ COSA SIGNIFICA "METTERE ONLINE" UN SITO
════════════════════════════════════════════════════════════

Ora il tuo sito è sul TUO computer (locale).
Per renderlo accessibile a chiunque su Internet devi:

┌─────────────────────────────────────────────────────────┐
│ 1. CARICARE I FILE su un SERVER (hosting)              │
│ 2. Il server gli assegna un INDIRIZZO pubblico (URL)   │
│ 3. Chiunque può visitare quell'URL e vedere il sito!   │
└─────────────────────────────────────────────────────────┘

Esempio:
• ORA: file://C:/Users/gianl/nodebiti/index.html (solo tu)
• DOPO: https://nodebiti.netlify.app (tutto il mondo!)

─────────────────────────────────────────────────────────────
📝 METODO 1: NETLIFY DROP (IL PIÙ FACILE)
─────────────────────────────────────────────────────────────

PASSO 1: Apri Netlify Drop
   → Vai su: https://app.netlify.com/drop
   → (L'ho già aperto per te nel browser!)

PASSO 2: Trascina il file
   → Dalla cartella "nodebiti-ottimizzato" (aperta in Explorer)
   → Trascina il file: index-standalone.html
   → Rilascialo nella finestra del browser

PASSO 3: Aspetta 10 secondi
   → Netlify caricherà il file
   → Ti mostrerà una barra di progresso

PASSO 4: FATTO! 🎉
   → Vedrai il tuo link pubblico tipo:
     https://eloquent-curie-a7b9c2.netlify.app
   
   → COPIA questo link e condividilo!
   → Il sito è ONLINE e raggiungibile da CHIUNQUE!


💡 BONUS: Personalizza il nome (OPZIONALE)
   Se ti registri gratuitamente su Netlify puoi cambiare il nome:
   
   Da: https://eloquent-curie-a7b9c2.netlify.app
   A:  https://nodebiti-demo.netlify.app

3️⃣ OPZIONE B: DOMINIO PERSONALIZZATO
════════════════════════════════════════════════════════════

✅ PERFETTO PER:
   - Siti professionali/aziendali
   - Branding (www.nodebiti.it)
   - Progetti commerciali
   - Maggiore credibilità

💰 COSTO: €10-20/anno (per il dominio)
⏱️ TEMPO: 15-30 minuti primo setup
🔗 RISULTATO: https://www.nodebiti.it


─────────────────────────────────────────────────────────────
PROCESSO COMPLETO: SITO + DOMINIO PERSONALIZZATO
─────────────────────────────────────────────────────────────

🔵 FASE 1: Verifica Disponibilità Dominio
─────────────────────────────────────────────────────────────

1. Vai su uno di questi siti:
   • Aruba.it (italiano, facile)
   • Register.it (italiano)
   • Namecheap.com (internazionale, economico)
   • GoDaddy.com (più conosciuto)

2. Cerca "nodebiti.it" nella barra di ricerca

3. Risultati possibili:
   ✅ DISPONIBILE → Puoi comprarlo!
   ❌ GIÀ PRESO → Prova varianti:
      - nodebiti.com
      - no-debiti.it
      - gestione-debiti.it
      - nodebiti.org

4. Prezzo tipico:
   • .it → €8-15/anno
   • .com → €10-15/anno
   • .org → €12-18/anno


🔵 FASE 2: Acquista il Dominio
─────────────────────────────────────────────────────────────

CONSIGLIO: Usa ARUBA o REGISTER (italiani, semplici)

PASSO 1: Aggiungi il dominio al carrello

PASSO 2: Scegli la durata:
   • 1 anno → ~€10
   • 2 anni → ~€18 (risparmi)
   • NON serve hosting da loro!

PASSO 3: Privacy WHOIS (IMPORTANTE!)
   ✅ Attiva la protezione privacy
   → Nasconde i tuoi dati personali pubblici

PASSO 4: Completa l'acquisto
   • Carta di credito / PayPal
   • Riceverai email di conferma

⏱️ TEMPO: Il dominio sarà attivo in 1-24 ore


🔵 FASE 3: Pubblica il Sito (Hosting)
─────────────────────────────────────────────────────────────

OPZIONE 3A: Netlify (CONSIGLIATO - Gratis!)
───────────────────────────────────────────────

1. Crea account su https://www.netlify.com
   (gratis con email/GitHub)

2. Carica il sito:
   • Trascina i file (come Netlify Drop)
   • OPPURE collega GitHub repository

3. Il sito sarà su: https://tuo-sito.netlify.app


OPZIONE 3B: Vercel (Alternativa eccellente)
───────────────────────────────────────────────

1. Registrati su https://vercel.com

2. Deploy progetto (drag & drop)

3. Sito su: https://nodebiti.vercel.app


OPZIONE 3C: Hosting Aruba (se hai comprato dominio lì)
───────────────────────────────────────────────

1. Aggiungi hosting al tuo account Aruba (~€20/anno)

2. Carica file via FTP/FileManager

3. Sito direttamente su www.nodebiti.it


🔵 FASE 4: Collega Dominio → Sito
─────────────────────────────────────────────────────────────

Ora hai:
✓ Dominio (es: nodebiti.it) su Aruba
✓ Sito online su Netlify (es: nodebiti.netlify.app)

Devi COLLEGARE i due!


SU NETLIFY:
───────────

1. Vai su Netlify Dashboard > Domain Settings

2. Clicca "Add custom domain"

3. Inserisci: nodebiti.it

4. Netlify ti darà dei DNS da configurare, tipo:
   apex-loadbalancer.netlify.com


SU ARUBA (Pannello Domini):
───────────

1. Vai su "Gestione DNS"

2. Aggiungi record:

   Tipo   Nome    Valore
   ────────────────────────────────────────
   A      @       75.2.60.5 (IP Netlify)
   CNAME  www     nodebiti.netlify.app


3. Salva

⏱️ Propagazione DNS: 1-48 ore (di solito 2-6 ore)


🔵 FASE 5: HTTPS (Certificato SSL)
─────────────────────────────────────────────────────────────

Su Netlify/Vercel:
✅ AUTOMATICO! Attivato gratis in pochi minuti
   → Il tuo sito avrà https:// (lucchetto verde)


Su Aruba Hosting:
1. Pannello controllo > SSL
2. Attiva "Let's Encrypt" (gratis)
3. Fatto!
